The modules described in this chapter provide a wide range of string
manipulation operations and other text processing services.

The |:library/codecs.txt/module-codecs:codecs| module described under |:library/binary.txt/binaryservices:Binary Data Services| is also highly
relevant to text processing. In addition, see the documentation for Python’s
built-in string type in |:library/stdtypes.txt/textseq:Text Sequence Type — str|.
